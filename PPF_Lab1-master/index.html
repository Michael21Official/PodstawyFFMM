Projektowanie i programowanie front-end

laboratorium 1: git

Przykadowe uycie standardowych znaczników html

Praca z git:

  • Wykonaj i zapisz zmiany w kodzie projektu
  • Przejdź do panelu "Source COntrol"
  • Ustaw zmodyfikowanym plikom stan "Straged"
  • Wprowadź opis do pola tekstowego Message
  • Zatwierdź zmiany lokalnie (commit)
  • Pobierz i scal zmiany zdalne (pull)
  • Rozwiąż ewentualne problemy
  • Wyślij aktualną wersję na serwer (push)
